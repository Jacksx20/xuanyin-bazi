## 📖 项目简介

玄音八字是一个**完全自包含**的八字命理分析引擎，无需外部MCP服务，所有计算由内置Python脚本完成。严格遵循传统八字命理学规则，确保计算结果的准确性和权威性。

### ✨ 核心特点

- ✅ **精准计算**：严格遵循传统八字规则，考虑立春、节气交接
- ✅ **完全自包含**：无需外部依赖服务，离线可用
- ✅ **功能完整**：八字排盘、大运推算、命理分析、黄历查询
- ✅ **算法权威**：五虎遁月、五鼠遁时、节气精确判断
- ✅ **易于使用**：命令行接口 + 自然语言对话

## 🔧 核心算法

### 年柱计算
- **基准年**：公元4年为甲子年
- **立春分界**：立春前属上一年，立春及以后属本年
- **计算公式**：`year_gan = (year - 4) % 10`, `year_zhi = (year - 4) % 12`

### 月柱计算
- **五虎遁月法**：根据年干推算月干基准
  - 甲己年丙寅起，乙庚年戊寅起
  - 丙辛年庚寅起，丁壬年壬寅起
  - 戊癸年甲寅起
- **节气交接**：节气日及以后属新月份
  - 立春(2月4日) → 寅月
  - 惊蛰(3月6日) → 卯月
  - 清明(4月5日) → 辰月
  - 以此类推...

### 日柱计算
- **基准日**：1900年1月1日为甲戌日
- **计算方法**：距离基准日的天数偏移
- **六十甲子**：严格遵循60日循环

### 时柱计算
- **五鼠遁时法**：根据日干推算时干基准
  - 甲己日起甲子，乙庚日起丙子
  - 丙辛日起戊子，丁壬日起庚子
  - 戊癸日起壬子
- **时辰划分**：
  - 子时：23:00-01:00
  - 丑时：01:00-03:00
  - 寅时：03:00-05:00
  - 以此类推...

## 🚀 使用示例

### 命令行使用

```bash
# 1. 八字排盘
python scripts/main.py --action bazi --gender 1 --datetime "1990-06-15T12:00:00+08:00"

# 2. 黄历查询（今日）
python scripts/main.py --action huangli

# 3. 黄历查询（指定日期）
python scripts/main.py --action huangli --datetime "2024-02-10T00:00:00+08:00"

# 4. 八字反查
python scripts/main.py --action reverse --bazi "庚午 壬午 辛亥 甲午"

# 5. 完整命理分析报告
python scripts/main.py --action report --gender 1 --datetime "1983-04-07T10:00:00+08:00" --output report.md
```

### 自然语言对话（安装Skill后）

```
用户：帮我排个八字，男，1990年6月15日中午12点

助手：[输出今日黄历宜忌]

用户：八字庚午壬午辛亥甲午对应什么时间？
助手：[输出反查结果]
```
