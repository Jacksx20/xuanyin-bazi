# 执行流程

## 一句话流程

```
解析用户输入 → 运行脚本(--action report) → 原样呈现Markdown报告
```

## Step 1：解析用户输入

从用户消息中提取性别和出生时间，转为脚本参数：

| 用户说法 | 转换 |
|----------|------|
| "男/女" | `--gender 1/0` |
| "1990年6月15日中午12点" | `--datetime "1990-06-15T12:00:00+08:00"` |
| "下午3点" | 小时=15 |
| "凌晨2点" | 小时=2 |

## Step 2：运行脚本

```bash
python {{current_skill_dir_path}}/scripts/main.py --action report --gender <0或1> --datetime "<ISO时间>"
```

脚本直接输出完整Markdown报告，包含：基础信息、四柱详析、五行统计、日主旺衰、格局、性格、事业、财运、感情、健康、神煞、刑冲合会、大运、当前运势。

## Step 3：呈现结果

直接将脚本输出的Markdown呈现给用户，无需AI二次加工或解读。

## 异常处理

| 异常 | 处理 |
|------|------|
| 时间格式错误 | 提示正确格式，给出示例 |
| 时辰不确定 | 提示时辰对时柱影响，建议尽量精确 |
| 脚本调用失败 | 检查Python环境和依赖是否安装 |
| 八字反查无结果 | 提示该八字组合可能不存在或超出搜索范围 |
