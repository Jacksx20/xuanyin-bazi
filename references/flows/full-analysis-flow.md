# 完整分析流程

## 流程概览

```
需求采集 → 数据获取 → 日主旺衰 → 格局判定 → 多维解读 → 大运分析 → 结果呈现
```

## Step 1：需求采集

### 1.1 确认分析类型

询问用户需要哪种分析：

| 选项 | 说明 |
|------|------|
| A. 八字排盘 | 仅排盘+基础解读 |
| B. 综合命理 | 全维度深度解读 |
| C. 黄历查询 | 查看某日宜忌 |
| D. 八字反查 | 已知八字反查时间 |
| E. 运势速查 | 当前运势+近期流年 |

### 1.2 收集出生信息

**必要信息**：
- 性别（男/女）
- 出生时间（公历或农历，精确到时辰）

**可选信息**：
- 早晚子时配置（默认选2，即23-24点日干支取当日）
- 出生地点（用于真太阳时校正，目前脚本不支持，仅作记录）

### 1.3 时间格式转换

根据用户输入转换为脚本所需格式：

| 用户输入 | 转换为 |
|----------|--------|
| "1990年6月15日中午12点" | `--datetime "1990-06-15T12:00:00+08:00"` |
| "1990-06-15 12:00" | `--datetime "1990-06-15T12:00:00+08:00"` |

## Step 2：数据获取

### 2.1 调用计算脚本

```bash
python {{current_skill_dir_path}}/scripts/main.py \
  --action bazi \
  --gender <0女/1男> \
  --datetime "<ISO时间>" \
  --early-zi <1或2>
```

### 2.2 数据提取

从返回结果中提取关键信息：

| 数据 | 路径 | 用途 |
|------|------|------|
| 八字 | `result.八字` | 基础展示 |
| 日主 | `result.日主` | 旺衰判断 |
| 四柱十神 | `result.年柱/月柱/日柱/时柱.天干.十神` | 格局/性格/事业 |
| 地支藏干 | `result.各柱.地支.藏干` | 格局判定/刑冲合会 |
| 神煞 | `result.神煞.年柱/月柱/日柱/时柱` | 神煞解读 |
| 大运 | `result.大运.大运[]` | 大运走势 |
| 刑冲合会 | `result.刑冲合会` | 关系解读 |
| 胎元命宫 | `result.胎元/胎息/命宫/身宫` | 延伸信息 |

## Step 3：日主旺衰判断

### 3.1 统计五行力量

1. **日干五行**：确定日主（如"辛"→金）
2. **月令五行**：月支主气五行（如"午"→火），判断得令
3. **四柱五行统计**：统计天干+地支藏干的五行分布
4. **生扶力量**：生我+同我的力量总和
5. **克泄力量**：克我+我生+我克的力量总和

### 3.2 判断旺衰

| 条件 | 结论 |
|------|------|
| 得令 + 生扶 > 克泄 | 偏旺/太旺 |
| 失令 + 克泄 > 生扶 | 偏弱/太弱 |
| 生扶 ≈ 克泄 | 中和 |

### 3.3 确定喜用忌仇

| 日主状态 | 喜神 | 用神 | 忌神 | 仇神 |
|----------|------|------|------|------|
| 偏旺 | 克我(官杀) | 泄我(食伤) | 生我(印星) | 同我(比劫) |
| 偏弱 | 生我(印星) | 同我(比劫) | 克我(官杀) | 泄我(食伤) |

## Step 4：格局判定

### 4.1 取格

1. 查看月令地支藏干
2. 藏干透于天干者取为格
3. 多干透出取力量最强者为格
4. 无透干取月令本气为格

### 4.2 格局成败

- **成格**：格神有根+无破格之物 → 格局成立
- **破格**：格神受克合+破格之物有力 → 格局破败，降低一格论

## Step 5：多维解读

按以下顺序逐一解读，每个维度引用 [interpretation-guide.md](../guides/interpretation-guide.md) 对应章节：

1. **性格解读** → 日主五行 + 十神分布 + 神煞特色
2. **事业解读** → 十神取象 + 五行取行业 + 格局定方向
3. **财运解读** → 正偏财分布 + 食伤生财 + 比劫夺财 + 财库
4. **感情解读** → 日支十神 + 财官分布 + 桃花 + 日支刑冲
5. **健康解读** → 五行偏枯 + 对应脏腑 + 刑冲影响

## Step 6：大运分析

### 6.1 大运吉凶标注

遍历每步大运，判断：

1. 大运天干十神 → 喜用/忌神
2. 大运地支与命局 → 有无冲合
3. 综合判断吉凶

### 6.2 关键年份标注

1. 大运交脱年（起运年+每10年）
2. 流年与命局天克地冲的年份
3. 流年与命局天合地合的年份

### 6.3 当前运势

1. 确定当前所在大运
2. 结合当前流年干支
3. 分析大运+流年+命局的交互关系

## 异常处理

| 异常 | 处理 |
|------|------|
| 时间格式错误 | 提示正确格式，给出示例 |
| 时辰不确定 | 提示时辰对时柱影响，建议尽量精确 |
| 脚本调用失败 | 检查Python环境和依赖是否安装 |
| 八字反查无结果 | 提示该八字组合可能不存在或超出搜索范围 |
