# 命理解读指南

## 总则

1. **多维度交叉**：任何结论需至少两个维度支撑，忌单一指标断论
2. **辩证看待**：吉中有凶、凶中有吉，十神无绝对好坏
3. **格局为先**：先定格局再论细节，格局决定解读方向
4. **动态视角**：大运流年可改变命局平衡，静态解读需标注"原局如此"
5. **语言规范**：用"倾向""可能""宜"等柔性表述，避免"一定""必然"等绝对论断

## 日主旺衰判断

### 判断方法

1. **得令**：日干生于当令季节（如甲木生于寅卯月）→ 旺
2. **得地**：日干在地支有根（长生/帝旺/建禄/羊刃）→ 旺
3. **得助**：四柱中比劫多 → 旺
4. **得生**：四柱中印星多 → 旺

### 旺衰等级

| 等级 | 特征 | 喜忌方向 |
|------|------|----------|
| 太旺 | 得令+得地+得助+得生 | 喜克泄耗，忌生扶 |
| 偏旺 | 得令+得地或得助 | 喜克泄耗，忌生扶 |
| 中和 | 得令或得地，克泄适中 | 平衡为上 |
| 偏弱 | 失令+克泄多 | 喜生扶，忌克泄耗 |
| 太弱 | 失令+无根+克泄重 | 喜生扶，忌克泄耗 |

### 解读话术

- 太旺："日主**偏旺**，自身气势强盛，宜以克泄耗为用，取食伤泄秀或财官制衡"
- 偏弱："日主**偏弱**，根基不足，宜以生扶为用，取印星生身或比劫助身"
- 中和："日主**中和**，命局平衡度好，行运适应面广"

## 格局解读

### 定格原则

1. 以月令地支藏干为格
2. 月令藏干透于天干者取为格
3. 多干透出取力量最强者为格
4. 月令无藏干或藏干不透，取对日主影响最大者为格

### 常见格局特征

| 格局 | 性格 | 事业方向 | 注意事项 |
|------|------|----------|----------|
| 正官格 | 正直守纪、循规蹈矩 | 公务员、管理、法律 | 忌伤官破格 |

| 偏印格 | 聪慧孤僻、偏才突出 | 技术、玄学、艺术 | 忌食神被枭夺 |
| 食神格 | 温和有福、才华内敛 | 文艺、餐饮、教育 | 忌枭印夺食 |
| 伤官格 | 才华横溢、叛逆创新 | 技术、表演、自由业 | 忌官星被伤 |

## 财运解读

### 解读框架

1. **正偏财分布**：正财→穏定收入、偏财→投资横财
2. **食伤生财**：食伤为财源，食伤旺则生财有道
3. **财库**：辰戌丑未为四库，财入库则能守
4. **比劫夺财**：比劫多则破财，需官杀制比劫护财
5. **大运财星**：行财运→财运亨通、行比劫运→破财之象

### 财运类型判断

| 条件 | 财运类型 | 建议 |
|------|----------|------|
| 正财旺+食伤生 | 穏定丰厚 | 宜穏健投资 |
| 偏财旺+伤官生 | 投机取财 | 宜短线操作，忌贪 |
| 财多身弱 | 富屋贫人 | 宜合作求财，不宜独担 |
| 身旺财弱 | 有力无财 | 待财运到时发力 |
| 比劫夺财 | 破财之象 | 宜守不宜攻，防合伙纠纷 |

## 健康解读

### 五行与脏腑

| 五行 | 脏(阴) | 腑(阳) | 过旺致病 | 过弱致病 |
|------|--------|--------|----------|----------|
| 木 | 肝 | 胆 | 肝气郁结、头痛眩晕 | 肝血不足、视力下降 |
| 火 | 心 | 小肠 | 心火旺盛、失眠烦躁 | 心血不足、面色苍白 |
| 土 | 脾 | 胃 | 脾胃湿热、消化不良 | 脾胃虚弱、食欲不振 |
| 金 | 肺 | 大肠 | 肺热咳嗽、皮肤干燥 | 肺气不足、易感冒 |
| 水 | 肾 | 膀胱 | 肾阳过亢、水肿 | 肾气不足、腰膝酸软 |

### 解读原则

1. 找出命局中**最旺**和**最弱**的五行
2. 最旺五行→过旺致病风险
3. 最弱五行→过弱致病风险
4. 刑冲导致五行失衡→对应脏腑受影响
5. 大运流年加重某五行→该五行对应脏腑需注意

## 大运解读

### 解读框架

1. **起运年龄**：几岁开始行大运
2. **每步大运**：干支+十神+年龄区间+起止年份
3. **吉凶判断**：大运为喜用→吉、大运为忌神→凶
4. **关键年份**：大运交脱年、流年与命局形成冲合的年份

### 大运吉凶判断

| 情形 | 吉凶 | 说明 |
|------|------|------|
| 大运为喜用神 | 吉 | 事业顺利、财运亨通 |
| 大运为忌神 | 凶 | 事业受阻、破耗多端 |
| 大运与命局天合地合 | 大吉 | 万事顺遂 |
| 大运与命局天克地冲 | 大凶 | 变动剧烈、灾厄风险 |
| 大运与流年相同（伏吟） | 凶 | 反复不安、事倍功半 |
| 大运与流年天克地冲（反吟） | 大凶 | 大起大落、悲事连连 |

## 刑冲合会解读

### 解读原则

1. **冲**：变动、冲突、分离——主动力，不可忽视
2. **合**：融合、牵绊、合作——合住则不动
3. **刑**：内耗、折磨、是非——暗流涌动
4. **害**：暗损、阻碍、不睦——小人是非
5. **暗合**：暗中勾连、隐秘关系——表面不见

### 冲的位置意义

| 冲的位置 | 影响 |
|----------|------|
| 年月冲 | 祖辈/父母宫不稳，早年奔波 |
| 月日冲 | 父母/兄弟与自身不合，事业变动 |
| 日时冲 | 配偶/子女宫不稳，中年变动 |
| 年时冲 | 祖辈与子女缘薄 |
